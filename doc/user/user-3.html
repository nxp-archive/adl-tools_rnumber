All off the rnumber functions start with the prefix rnumber_. Most of these functions take a pointer to a rnumber as their first argument and return a pointer to a newly allocated pointer to an rnumber. The deviant functions (ie. the ones who do not return a newly allocated rnumber) are listed separately below.
