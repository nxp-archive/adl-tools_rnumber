All of the c functions can be called from perl by using the rnumber module (see examples). Several of the functions are exported for convenience. Also, some of the functions have be made polymorphic. They can accept rnumbers or integers as any of their parameters. They are also listed here.

5.1 Memory Management

Do not call rnumber_destroy, Perl will call it for you.

rn_int(rnumber)convert rnumber to integer (may cause truncation)
rn_eq(scalar, scalar)are integers or rnumbers equal.
rn_neq(scalar, scalar)are integers or rnumbers not equal.
rn_lt(scalar, scalar)is first variable less than second (integers or rnumbers)
rn_le(scalar, scalar)is first variable less than or equal to second (integers or rnumbers)
rn_gt(scalar, scalar)is first variable greater than second (integers or rnumbers)
rn_ge(scalar, scalar)is first variable greater than or equal to second (integers or rnumbers)
rn_ls(scalar, scalar)left shift first variable by second number of bits. (integers or rnumbers)
rn_rs(scalar, scalar)right shift first variable by second number of bits. (integers or rnumbers)
rn_plus(scalar, scalar)add first and second variables together and return an rnumber. Arguments may be either rnumbers or integers
rn_minus(scalar, scalar)subtrace second variable from first and return an rnumber. Arguments may be either rnumbers or integers
rn_div(scalar, scalar)divide first variable by second and return an rnumber. Arguments may be either rnumbers or integers
rn_mult(scalar, scalar)multiply first and second variables together and return an rnumber. Arguments may be either rnumbers or integers
rn_bitor(scalar, scalar)bitwise or first and second variables together and return an rnumber. Arguments may be either rnumbers or integers
rn_bitand(scalar, scalar)bitwise and first and second variables together and return an rnumber. Arguments may be either rnumbers or integers
rn_mod(scalar, scalar)mod first variable by second and return an rnumber. Arguments may be either rnumbers or integers
rn_size(rnumber)Return the number of bits in the RNumber passed as an argument.
